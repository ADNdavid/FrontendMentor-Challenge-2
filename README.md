### What I learned

- Use the properties **grid-template-areas** and **grid-area** to center a tag that contains the card.

```css
.container{
    display:grid;
    grid-template-areas:
        ". . ."
        ". card ."
        ". . .";
    }
.item {
    grid-area: card;
    }
```

- Use **Semantic Tags** in HTML structure.
